EU Migration Dashboard

Interactive visualization of EU intra-migration patterns

How to Embed This Dashboard

To embed this dashboard on your website, use the following iframe code:

<iframe src="https://eu-dashboard.stellux.org/?embedded=true" width="100%" height="600px" frameborder="0" allowfullscreen> </iframe>

Embedding Options

  • Full Dashboard: https://your-domain.com/?embedded=true
  • Compact Mode: https://your-domain.com/?embedded=true&compact=true
  • Specific Country: https://your-domain.com/?embedded=true&country=DE
  • Specific Year: https://your-domain.com/?embedded=true&year=2020

Responsive Embedding

For responsive embedding, use CSS to make the iframe responsive:

.embed-container { position: relative; width: 100%; height: 0; padding-bottom: 75%; /* 4:3 aspect ratio */ } .embed-container iframe { position: absolute; top: 0; left: 0; width: 100%; height: 100%; }
